# HURE_admin_frontend

Hanyang Univ. Graduate School of Real Estate Convergence Reunion app frontend for Admin  
한양대학교 부동산융합대학원 원우회 앱 frontend for Admin

## How to launch server

```shell
yarn install
yarn start
```
